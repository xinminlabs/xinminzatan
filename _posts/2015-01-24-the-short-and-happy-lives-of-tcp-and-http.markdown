### 介绍

这是 Starr Horne 在 Golden Gate Ruby Conference 2014
做的主题分享，从网络协议上详解了为什么创建一个 tcp 或者 http
请求是代价高昂的，包括

* 网络延迟
* 握手协议
* 网络拥塞管理

演讲中也给出了可以优化的选项

* Web 服务器
* Linux 内核调优
* 减少 http 请求

演讲中给出的一些参考资料也非常值得一看。
