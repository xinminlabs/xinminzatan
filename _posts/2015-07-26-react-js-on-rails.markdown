### 介绍

这是 Michael Chan 在 RailsConf 2015
做的主题分享，介绍了 Rails 项目中加入 React.js 框架。

作者使用 react-rails gem，将 rails partial view 替换为
react_component，通过 http 请求获取数据，还介绍了一些使用 react
的技巧，对于想要在 rails 项目中尝试 react.js 的童鞋，还是值得一看的。
